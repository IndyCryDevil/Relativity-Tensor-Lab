Relativity Tensor Lab

A symbolic General Relativity tensor calculator built using FastAPI, SymPy, Next.js, and TypeScript.

Relativity Tensor Lab allows users to input spacetime metrics and compute important geometrical quantities used in General Relativity and Cosmology.

Installation
Clone Repository
git clone https://github.com/IndyCryDevil/Relativity-Tensor-Lab.git
Backend Setup
cd backend
python -m venv venv
venv\\Scripts\\activate
pip install -r requirements.txt
uvicorn main:app --reload

Backend runs on:

http://127.0.0.1:8000
Frontend Setup
cd frontend
npm install
npm run dev

Frontend runs on:

http://localhost:3000
